Fox Hunting is a logical game first published in USSR's scientific magazine in 1985 by A. Neschetny. Rules of the game are quite simple - there are 9 foxes hidden on he 10 x 10 play grid and your goal is to find all of them. By choosing any tile on the grid you can find a fox hiding under it, or you are provided with a number stating how many foxes there are on the same line with the given tile. The faster you find all the foxes and the less moves you make the better.

The idea is to build public API which can be used to implement clients for desktop and mobiles devices. The best approach would be to place it in a cloud to provide online multiplayer gaming for clients, with public high score boards.

In addition to the API it is also planned to implement at least one client, probably for Android devices.

FAQ

Is there a single player mode?
Yes, there is. You can play alone like in Minesweeper, or you can play with someone like in Battleship.
Are there any existing implementations of this game?
Yes, there are some online implementations, as well as few amateur desktop implementations, and there is at least one for Android.
Why do we need another implementation of this game?
Well, mine will be the best. :) And seriously, as I know there are no implementations with multiplayer support. The idea is to make it possible for users with different client to play with each other.
